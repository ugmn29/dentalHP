## 🎯 画像配置の流れ

1. 上記のディレクトリ構造に従って、`public/images/kidsortho/` 配下にフォルダを作成
2. 各画像ファイルを該当するフォルダに配置
3. ファイル名は **完全一致** させる（大文字小文字も含めて）
4. 拡張子は `.jpg` で統一（現在の設定）

## 💡 仮画像の表示内容

実際の画像が配置されるまで、各画像エリアには以下のように表示されます：

```
┌─────────────────────────────┐
│                             │
│      【準備中】             │
│   underbite-hero.jpg        │
│                             │
│ 配置先: /public/images/     │
│   kidsortho/types/          │
└─────────────────────────────┘
```

画像ファイルを配置すると、自動的に実画像が表示されます。
